# 7-Day Intensive Crash Course  
**Goal:** Refresh & master Node.js + React + DS/Algo + Web Security + Design Patterns + AWS Basics  
**Daily commitment:** 4 focused hours/day  
**Total:** ~28 hours  
**Date range:** [Insert your start date] → +7 days  
**Philosophy:** 60–70% hands-on coding, heavy project building, no mercy

## General Setup (Do before Day 1 – 10-15 mins)
- Node.js LTS + npm
- VS Code + useful extensions (ESLint, Prettier, React snippets, Thunder Client)
- Git & fresh GitHub repo for daily commits
- AWS Free Tier account (sign up & verify)
- Postman / Insomnia for API testing

## Daily Structure (4 hours)
- Hour 1 → Theory / Reading / Videos
- Hours 2–3 → Intense coding + exercises
- Hour 4 → Review + mini-project / integration / refactor

## Day 1 – Node.js Fundamentals + DS/Algo Basics
- Node.js: Event loop, modules, npm, async/await, promises
- Build: Simple CLI task manager (file I/O)
- DS: Arrays, Linked Lists, Stacks, Queues (implement in JS)
- LeetCode Easy: Two Sum, Valid Parentheses, Reverse Integer (5–7 problems)
- Review: Big O notation + use cases

## Day 2 – Node.js Servers + Algo Search/Sort
- Express.js: Routing, middleware, REST APIs, error handling
- Build: Basic CRUD Todo REST API
- Algo: Binary Search, Quick Sort, Merge Sort
- LeetCode: Search Insert Position, Sort Colors, Contains Duplicate II (6–8 mediums)
- Integrate: Sorting endpoint in your API

## Day 3 – React Fundamentals + Structural Design Patterns
- React: Components, JSX, props, state, hooks (useState, useEffect)
- Build: Todo list UI (Create React App)
- Patterns: Singleton, Factory, Module (implement in JS)
- Connect: Fetch data from Day 2 Node API
- Review: OOP principles in React context

## Day 4 – React Advanced + Web Security Fundamentals
- React: Context API, React Router, memo/useMemo, basic performance
- Security: XSS prevention, CSRF, Helmet.js, secure headers, input sanitization
- Enhance: Add routing + global state to previous app
- Practice: Manual injection attacks & fixes
- Study: OWASP Top 5 (Injection, Broken Auth, XSS, Insecure Deserialization, Security Misconfig)

## Day 5 – Advanced DS/Algo + Behavioral Design Patterns
- DS: Binary Search Trees, Tree traversals, Graphs (DFS/BFS basics)
- LeetCode Medium: Invert Binary Tree, Number of Islands, Course Schedule (7–9 problems)
- Patterns: Observer, Strategy, Decorator (apply in React/Node)
- Refactor: Use patterns in existing full-stack todo app

## Day 6 – AWS Basics + Full-Stack Deployment
- AWS: EC2, S3, Lambda, IAM basics, security groups
- Deploy: Node API → Lambda / EC2, React build → S3
- Practice: AWS CLI basics (configure, s3 cp, etc.)
- Secure: Enable HTTPS (ACM or CloudFront)
- Review: Full request flow (React → API → AWS)

## Day 7 – Synthesis, Polish & Interview Prep
- Security deep dive: OWASP Top 10 quick review
- Build: Final polished secure Todo app (React + Node + AWS)
- LeetCode mixed: 5 medium problems (any topic from week)
- Mock interview prep:
  - Explain 3 design patterns with use cases
  - Common security vulnerabilities & fixes
  - AWS deployment choices & trade-offs
- Polish GitHub: Clean README, commit history, live demo link (if possible)
